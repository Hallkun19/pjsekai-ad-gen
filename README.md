# pjsekai-ad-gen
プロセカ風宣伝画像ジェネレーター

# 使い方
Releaseから最新版をダウンロードしちゃってください。ちなみに私はgithubをほとんど使ったことがないため、おそらくミスが多発しますが許してください。
ちなみに二次配布はOKです。ぶっちゃけ制作者が私であることを明記してくれさえすればそれでいいです。
